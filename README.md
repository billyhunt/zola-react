# zola-react

## zola take home assignment to implement a sortable, filterable grid using react



To install, run 

### 'npm install`


To start, run:
### `npm start`


Runs the app in the development mode.<br>
Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
